## About the Project
This project analyzes customer retention and churn behavior using real-world
subscription data. The objective is to identify why customers leave and what
strategies can improve long-term customer retention.

## Business Questions
- Why are customers leaving the platform?
- Which customer segments have the highest churn?
- How does customer tenure impact retention?
- What actions can reduce customer churn?

## Dataset Used
Telco Customer Churn Dataset (Kaggle)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Churn Rate Analysis
- Data Visualization

## Key Insights
- Month-to-month contract customers have the highest churn
- Churn is highest in the first 6 months of customer lifecycle
- Electronic check payment users show higher churn
- Long-term contracts significantly improve retention

## Business Recommendations
- Encourage long-term contracts with incentives
- Improve onboarding during the first 60 days
- Promote automatic payment methods
- Identify high-risk customers early

## Conclusion
This analysis provides actionable insights that help subscription-based
businesses reduce churn and improve customer lifetime value.
